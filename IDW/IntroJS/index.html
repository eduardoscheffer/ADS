Introdução ao JavaScript - Lista de exercícios

1 - Crie um script que leia dois números e apresenta a multiplicação deles.

2 - Faça um script que leia um número e mostre uma mensagem indicando se este número é par ou ímpar e se é positivo ou negativo.

3 - Faça um script que leia o nome e a idade de uma pessoa e informe se a pessoa é maior de idade ou não. Exemplo: ‘João você tem 30 anos, é maior de idade’.

4 - Faça um script que receba um número, calcule e mostre a tabuada desse número.

5 - Crie um script que leia três notas e imprima a média das notas e informe se o aluno está aprovado ou não. A média para aprovação é 6.

6 - Crie um novo script que leia um número que só pode ser de 1 a 7 e imprima os dias da semana sendo 1 = domingo e 7 = sábado.

7 - Faça um script que apresenta a soma dos números múltiplos de 3 entre 0 e 100.

8 - Faça um script que leia números até que seja digitado um número negativo, ao final mostrar a média dos números digitados.
